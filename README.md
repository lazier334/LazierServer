# LazierServer
一个可以快速搭建的服务器

# 核心技术栈
**Brain + Koa + Vue + NaiveUI**

# 文件夹说明
```c
LazierServer
├─ bin          // 脚本工具
├─ public       // 公开的资源
├─ ld           // 本地数据、配置、插件等 
├─ src          // 代码
│  ├─ app.js    // 主程序
│  ├─ libs      // 库、包
│  └─ plugins   // 插件
└─ tests        // 测试项
```

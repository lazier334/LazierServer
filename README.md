# LazierServer
一个可以快速搭建的服务器

# 基于 [LocalServer](#lazierserver) 改进的一个可以快速搭建伪站点的服务器

# 核心技术栈
**`Brain` + `Koa` + `Vue` + `NaiveUI`**

# 文件夹说明
```c
LazierServer
├─ bin          // 脚本工具
├─ public       // 公开的资源
├─ web          // 项目资源站
├─ webScript    // 插件资源站
├─ ld           // 本地数据、配置、插件等 
├─ src          // 代码
│  ├─ app.js    // 主程序
│  ├─ libs      // 库、包
│  └─ plugins   // 插件
└─ tests        // 测试项
```
